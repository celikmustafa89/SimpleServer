WebServer Program(R) Version 1.1  10/03/2014

CONFIGURATION INSTRUCTION
-------------------------
- WebServer varsıyalan olarak localhost:8080 portuna bağlanarak çalışmaktadır. config.properties dosyasından port(PORT=8080) numarası değiştirilebilir.
- programın logları varsayılan olarak "requestLogs.txt" ve "serverLogs.txt" dosyalarında tutulmaktadır. config.properties dosyasından loglama dosyalarının adı değiştirilebilir.

HOW TO RUN
----------
- Program çalıştırılır.
- Herhangi bir browserdan "localhost:8080/index.html" talebi yazıldıktan sonra istek görüntülenir.
- Bunun dışındaki taleplerde "404 File Not Found" fatası görüntülenir.
